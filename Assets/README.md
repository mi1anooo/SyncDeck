# SyncDeck Assets

## /Themes
Per-theme background images or SVG overlays.
File naming: `{ThemeId}_frame.png`, `{ThemeId}_bg.png`

## /Icons
App icon files:
- `app.ico`  (Windows — 256x256 ICO)
- `app.icns` (macOS  — generated from PNG)
- `app.png`  (base 512×512 PNG source)

## /Frames
Decorative frame overlays rendered over the visualizer area.
File naming: `{ThemeId}_frame_overlay.png`

## /MockAlbumArt
Static test album art images for MockMusicProvider development.
File naming: `mock_art_{1..6}.jpg`
